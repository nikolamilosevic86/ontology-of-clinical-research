usage:
java -jar HSDB_dme.jar <owl_file_url> <local_path_to_included_ontologies>

Example:

On Windows PC
$ java -jar C:/BMor/GLIF/Design_Patterns/Ida/OCRe_New_SVN/HSDB_data_model_extractor/HSDB_dme.jar file:///C:/BMor/GLIF/Design_Patterns/Ida/OCRe_New_SVN/ontology/HSDB_OCRe.owl C:/Dropbox/OCRe/source/ontology >C:/BMor/GLIF/Design_Patterns/Ida/OCRe_New_SVN/xsd/ERGO_Annotation.xsd 2>&1
to redirect the output to HSDB.xsd
